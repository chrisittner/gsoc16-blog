GSoC proposal accepted!

Posted on Tue 26 April 2016 in general

My proposal for Google Summer of Code 2016 has been accepted :). This means that I will spend part of my summer working on the
pgmpy library. I will implement some techniques for Bayesian Network structure learning. You can have a look an my proposal here.

As a first step, I set up this blog to document my progress. It is built with the Pelican static-site generator and hosted on GitHub pages.

Updates will follow!
